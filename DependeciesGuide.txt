This document will guide your through the installation process of this project:

Dependencies list:
1. Tkinter: usually go with your Python, how to check:
  a. Open your command line/terminal, type python to open Python console
  b. Type: import Tkinter # no underscore, uppercase 'T' for versions prior to V3.0 or
           import tkinter # no underscore, lowercase 't' for V3.0 and later
  c. Type: Tkinter._test() # note underscore in _test and uppercase 'T' for versions prior to V3.0 or
           tkinter._test() # note underscore in _test and lowercase 'T' for V3.0 and later
2. PIL: usually go with your Python
3. scikit-image: run setup.py to install
4. scikit-learn: run setup.py to install
5. numpy: run setup.py to install
6. matplotlib: run setup.py to install
7. nltk: run setup.py to install

If the setup.py run correctly, everything will be okay, you don't need to do anything,
if not, you need to install manually to get it works.

Follow the instructions in the link according to each packages
1. scikit-learn: http://scikit-learn.org/stable/install.html
2. matplotlib: http://matplotlib.org/users/installing.html
3. scikit-image: http://scikit-image.org/download.html
4. numpy: http://www.scipy.org/scipylib/download.html
5. nltk: http://www.nltk.org/install.html
